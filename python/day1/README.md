### Aditional Comments

1 - Print Modifiers

- in a string we can't do something like `print("She said: "Hello"  and then left")`
- we should do something like `print("She said: 'Hello' and left")`
- or another way is to use backslash `print("She said: \"Hello\" and then left)`
